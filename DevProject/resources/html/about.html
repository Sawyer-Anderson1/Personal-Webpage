About this Webpage:

Why:

This is my first real project in HTML and CSS. I initially learned about HTML, CSS, and JavaScript through CodeCademy, but had never really went beyond what was done within the courses of the site. This project will, I believe, give me lots of exposure, practice, and experience in HTML, CSS, JavaScript, and (maybe) some PHP. This is the main reason I have made this site, without templates of any sort. This website will continually be added upon and updated, i.e. it will never be "completed". Newly add features that will be implemented will be come more complex, and either for the purpose of practicality or artistic vision.

In addition to these practical and artistic features, I will also create a part(s) of the site for the purpose of another, entirely seperate project - that requires a webpage. I will document both the addition of features, and the ideas for furture ones on this page. The documentation of additions, if it changed an older part of the webpage in a significant way, will provided images and description of the old state of the site.

What:

First features:

  • The basic text and images (including this).
  • Buttons at the top bar that link to different sections of the home page, and to this about section.
  • Having the top bar stay when the user scrolls down.

Future features:

  • Dropdown menus for the top bar buttons that already exist. The dropdown would be for the buttons that have subsections.
  • Search box, maybe using datalist
  • Users being able to rate and give feedback, with both stars and comments
  • Implement different rendering methods: SSR, SPA, and SSG
  • Implement Database (needed for an effective storing of the feedback,etc.)
  • etc...
